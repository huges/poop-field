# Poop Field

**Strategic field navigation under uncertain biological conditions.**

An experimental decision-making simulator about uncertainty, intuition and survival.

Inspired by Minesweeper, but with higher stakes.

---

## Rules

- Left click reveals a cell  
- Right click places a **Poop flag**  
- Click a poop: Game Over  
- Click Start to begin again

---

## Core concepts explored

- probabilistic reasoning  
- risk management  
- false positives  
- catastrophic failure recovery

---

## Features

- Classic retro minefield interface  
- Biological hazard detection system  
- Probabilistic poop counter  
- Legacy-style threat visualization

---

## Tech

Vanilla HTML / CSS / JavaScript
